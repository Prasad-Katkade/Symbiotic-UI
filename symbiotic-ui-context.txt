рџ§  What We Are Building

SymbioticUI is an AI-native UI runtime layer for React Native (Expo + TypeScript) that allows:

runtime UI personalization
AI-driven layout mutations
long-press interaction editing
limited styling changes (Tailwind / NativeWind)
reorder / hide / show UI elements

Instead of treating JSX as source of truth, we convert UI into a:

SymTree (JSON-based UI graph)

and mutate it dynamically using AI-generated structured actions.

1. SymbioticUI.tsx
Purpose:

Main wrapper controller for Symbiotic system.

Responsibilities:
parses JSX в†’ SymTree
stores SymTree in state

Important Rules:

Only wrap static / layout-safe components
MUST maintain SymTree as state

2. parser.ts
Purpose:

Converts JSX into SymTree JSON.

Output Structure:

  id: string;
  type: string;
  props: Record<string, any>;
  children: SymNode[];
};
Responsibilities:
extract sym-id
flatten React children tree
assign stable IDs
remove runtime React references

Purpose:

Converts SymTree в†’ React Native UI

Responsibilities:

5. mutations.ts
